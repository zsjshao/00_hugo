## 数组

### 一维数组

数组定义

- 具有相同的数据类型
- 使用过程中需要保留原始数据

C语言为这些数据，提供了一种构造数据类型：数组。所谓数组就是一组具有相同数据类型的数据的有序集合。

一维数组的定义格式：

- 类型说明符  数组名[常量表达式];

例如：int a[10];

#### 访问越界

```
#include <stdio.h>
#include <stdlib.h>

#define N 5
int main()
{
  int i=10;
  int arr[N]={1,2,3,4,5};
  arr[5]=6;
  arr[6]=7;
  arr[7]=8;
  printf("i=%d\n",i);
}

[root@ali C]# ./a.out 
i=8
```

### 二维数组

二维数组定义的一般形式：

- 类型说明符  数组名 [常量表达式] [常量表达式]

#### 初始化

```
#include <stdio.h>
#include <stdlib.h>

int main()
{
  int a[3][4]={1,3,5,7,2,4,6,8,9,11,13,15};
  int b[3][4]={{1},{2},{9,11}};
}
```

#### 二维数组的传递

```
#include <stdio.h>
#include <stdlib.h>

void print(int arr[][4],int row)
{
  int i,j;
  for(i=0;i<row;i++)
  {
    for(j=0;j<sizeof(arr[0])/sizeof(int);j++)
    {
      printf("%3d",arr[i][j]);
    }
    printf("\n");
  }
}
int main()
{
  int a[3][4]={1,3,5,7,2,4,6,8,9,11,13,15};
  int b[3][4]={{1},{2},{9,11}};
  print(a,3);
}
```
